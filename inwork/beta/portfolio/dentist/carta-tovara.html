Модуль врача установки Contact, закреплен на трех шарнирном держателе, для пяти инструментов (с наличием фиброоптики), верхний подвод инструментов:

  • 1. для подключения электромотора шланг с фиброоптикой;
  • 2. для турбинного наконечника шланг с фиброоптикой;
  • 3. шприц Faro 3-х функциональный ;
  • 4. Блок для управления электроникой гидроблока и кресла пациента;
  • 5. Кнопки управления мембранные (срабатывают от простого прикосновения);
  • 6. Наличие коллектора для отработанной смазки из наконечников;
  • 7. Автодиагностика с начальным контролем
  • 8. Для 1-го инструментального столика Держатель
  • Модуль ассистента установки Contact с держателем для трех инструментов:
  • 1. пылесос;
  • 2. для подключения к автономной аспирационной системе слюноотсос со шлангами
  • 3. свободный модуль
  • 4. Сепарационная система с автоматикой METASYS
  • 5. Автономная система подачи жидкости на инструменты (дистиллированная вода, дез.раствор, физ.раствор) ABS

Вакуумную помпу можно не использовать благодаря сепаратору+автономная система подачи жидкости.

Кресло пациента установки Contact Lite Comfort S - программируемое электромеханическое, (4 запоминаемых позиций: нулевая позиция, две произвольных позиции, позиция полоскания и ):

  • 1. подголовник артикуляционный ;
  • 2. с левой стороны подлокотник ;
  • 3. с пульта врача и ножной педали происходит управление ;
  • 4. система защиты ног врача;
  • 5. покрытие антибактериальное ;
  • 6. четырех клавишная педаль для управления инструментами и креслом (плавное регулирование турбины, микромотора, реверс,)
